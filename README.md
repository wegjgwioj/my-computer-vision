# Video-stitching

## 一、问题背景

1. 多摄像头画面存在15-30%重叠区域，导致存储资源浪费和检索效率低下
2. 单镜头视角局限（通常<90°），无法满足场景的广域监控需求

通过开发实时视频拼接系统，实现：

- 360°全景监控画面实时合成
- 多路视频流智能去重处理
- 低延时高画质的监控回放
